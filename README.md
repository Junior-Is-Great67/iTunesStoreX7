# iTunesStoreX7
The ultimate fix for the iOS 7 AppStore! (Work in progress, bugs are expected)

# HOW TO INSTALL

## Cydia

1. Add the repo (cydia.jyvee8games.org) and grab the iTunesStoreX7 Tweak.

2. Restart the device after respringing

3. Log in and start your downloads!

NOTE: Due to the weird networking logic it WILL lag really bad. This will hopefully be optimised soon.

## Directly from the source

1. Get theos and run ```make packages```

2. Transfer the deb to your device.

3. Install it and restart the device.

4. Log in and start downloading your apps!

## Known Bugs

- The store will randomly use a ton of resources for an unknown reason

- iPads can sometimes crash the store when theres more than 1 app downloading at a time

- Storefronts are picky and will not fully be completed

- Some images just take forever to load as testing has seen
